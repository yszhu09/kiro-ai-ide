Installation Guide

Follow these simple steps to install Kiro on your Mac.

1

Download the DMG file

Click the download button above to get the appropriate version for your Mac (Intel or Apple Silicon).

2

Open the DMG file

Double-click the downloaded DMG file to mount the disk image. You'll see the Kiro application icon.

3

Install Kiro

Drag the Kiro application icon to your Applications folder. This will install Kiro on your Mac.

4

Launch Kiro

Open Kiro from your Applications folder or Spotlight search. You may need to allow it in Security & Privacy settings.

5

Get Started

Follow the welcome wizard to set up your development environment and start coding with AI assistance.
